# 📊 Forecast CSV RAG Chatbot

AI-powered chatbot that allows users to upload CSV files and ask natural language questions using RAG + Groq LLM.

## 🔗 Live Application
👉 **Try the App:** https://huggingface.co/spaces/Kavyaa/ChatBot_RAG

## 🧠 Features
- Upload CSV data
- Semantic search using embeddings
- RAG-based question answering
- Groq LLM integration
- Interactive Gradio UI
